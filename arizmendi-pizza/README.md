# Arizmendi Pizza Tracker

A mobile-friendly website to track Arizmendi Bakery's daily pizza schedule for the Inner Sunset location.

## Features

- 📱 Mobile-first, responsive design
- 🍕 Today's pizza highlighted prominently
- 📅 Next 14 days of pizza schedule
- 🌙 Dark mode support
- ⏰ Auto-refresh at midnight
- 🔗 Quick link to order online

## Usage

Simply open `index.html` in a browser. For local development:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx serve
```

Then open http://localhost:8000

## Updating Pizza Data

The pizza schedule is stored in `pizza-data.js`. Update this file monthly with the new schedule from [Arizmendi's Pizza Menu](https://www.arizmendibakery.com/pizza).
